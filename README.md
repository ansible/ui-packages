# Ansible UI Packages

## Getting Started

1. Prerequisites

   - Node 20.x and up
   - NPM 8.x and up

2. Clone Repository

   ```
   git clone git@github.com:ansible/ui-packages.git
   ```

3. Install Package Dependencies

   ```
   npm ci
   ```

4. Run the examples
   
   ```
   npm start
   ```

## Running Tests

Tests can be run form the CLI using

```
npm test
```
